Welcome to Nirvana System

The Nirvana System is a versatile communication framework designed to facilitate structured interactions among multiple agents. Developed by SummerMute, it supports both Hierarchical and Joint communication modes, enabling efficient collaboration and task management within teams. Leveraging AI Agents for natural language processing, the system ensures dynamic and meaningful exchanges among participants.

Background

The problem we identified is that current systems like DLMM or CLMM LP pools on Solana and in DeFi are highly complex and challenging for the average person to evaluate. LP pool investments are significantly more difficult than general trading, requiring individuals to monitor pools all day, precisely timing when to provide or withdraw liquidity. In reality, such high-difficulty trading demands the integration of various LMMs and the aggregation of diverse data sources to execute unbiased trading strategies effectively. To address this, we established an AI agency called SummerMute and developed an AI discussion system named Nirvana within it. This system enables AI agents to engage in deep discussions and use the resulting insights to trade LP pools effectively. Our system is designed to replace traditional market makers (MMs) in the future, maximizing LP pool efficiency and enabling markets to find balance more easily. It aims to become a reliable AI tool for sustainable and optimized liquidity management.

Overview

The Nirvana System is a flexible communication framework that supports both Hierarchical and Joint communication modes. By utilizing AI Agents for natural language processing, it ensures dynamic and meaningful interactions among team members, promoting effective collaboration and task management.

Features

  • Hierarchical Communication Mode: Designate a leader to manage and coordinate interactions among team members.
  • Joint Communication Mode (WIP): Foster equal participation and collaboration among all agents without a designated leader.
  • Dynamic Logging: Automatically generates log files with system type and timestamp for enhanced traceability.
  • Comprehensive Reporting: Creates detailed Markdown reports capturing all communication cycles and interactions.
  • TypeScript Integration: Ensures type safety and maintainability with well-defined interfaces and modular code structure.
  • Environment Configuration: Easily configurable through .env and JSON configuration files.
  • Scalable Architecture: Designed to handle increasing numbers of agents and complex communication flows.

Supported AI Agent Frameworks

  • ELIZA

Installation Guide

Prerequisites

Before setting up the Nirvana System, ensure you have the following installed on your machine:

  • Node.js: v23.3.0 (Recommended)
  • pnpm: v9.15.1 (Recommended)

Installation Steps

  1. Clone the Repository 
    git clone git@github.com/SummerMute-t/nirvana.git
cd nirvana
  2. Install Dependencies Using pnpm 
    pnpm install

Configuration

Environment Variables

Copy .env.sample to create a .env file in the root directory of the project.

.env

Base URL for the Agent (Eliza Framework)

ELIZA_BASE_URL=http://localhost:3000

Maximum number of communication cycles

MAX_COMMUNICATION_CYCLES=10

Agents Configuration

Define your agents and their roles in JSON configuration files. Two separate configurations are provided for Hierarchical and Joint modes. Refer to the sample configuration files in the groups/ directory for reference.

Run Nirvana System

To start the Nirvana System in Hierarchical Mode, execute the following command:

pnpm start --config groups/filename.json
